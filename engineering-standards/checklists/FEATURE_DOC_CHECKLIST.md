# Feature Doc Checklist

Use this checklist whenever a main flow changes.

- [ ] I changed feature behavior or flow.
- [ ] I updated the matching `FEATURE.md` in the same folder.
- [ ] I updated flow steps and state transitions.
- [ ] I updated error-handling notes if behavior changed.
- [ ] I added a short changelog line with date.

中文：每次主流程改动后，必须按上述清单同步更新文档。
