# Engineering Standards

This folder contains repository-level engineering constraints and templates.

## Files
- `FEATURE_DOC_POLICY.md`: mandatory feature documentation rules.
- `templates/FEATURE.template.md`: template for each feature folder doc.
- `checklists/FEATURE_DOC_CHECKLIST.md`: update checklist after flow changes.

中文：
本目录用于存放工程规范、模板与检查清单。
